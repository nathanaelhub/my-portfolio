## Overview

This project analyzes employee attrition patterns to help organizations develop effective retention strategies. Using Tableau for visualization, the analysis identifies key factors contributing to employee turnover and provides actionable insights.

## Key Objectives

- **Attrition Analysis**: Identify patterns and trends in employee turnover
- **Risk Factors**: Determine the key factors that contribute to employee attrition
- **Visualization**: Create interactive dashboards for stakeholder communication
- **Recommendations**: Develop data-driven retention strategies

## Dashboard Features

- **Interactive Filters**: Explore data by department, tenure, and demographics
- **Trend Analysis**: Visualize attrition trends over time
- **Risk Scoring**: Identify high-risk employee segments
- **Comparative Analysis**: Compare attrition across different organizational units

## Technologies Used

- **Tableau**: Primary visualization and dashboard tool
- **Python**: Data preprocessing and analysis
- **Statistical Analysis**: Correlation and regression analysis for factor identification

## Key Findings

The analysis revealed that factors such as work-life balance, career growth opportunities, and compensation significantly impact employee retention. The interactive dashboard enables HR teams to monitor and address attrition risks proactively.
