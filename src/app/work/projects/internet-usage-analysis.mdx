## Overview

This project explores global internet usage patterns and search engine preferences across different regions and demographics. The analysis provides insights into digital adoption trends and platform dominance worldwide.

## Key Objectives

- **Global Trends**: Analyze internet adoption rates across countries and regions
- **Search Engine Analysis**: Examine market share and preferences for different search engines
- **Demographic Patterns**: Understand how internet usage varies by age, income, and location
- **Trend Forecasting**: Project future internet adoption and usage patterns

## Dashboard Features

- **World Maps**: Interactive geographic visualization of internet penetration
- **Time Series**: Track growth trends over multiple years
- **Comparative Analysis**: Compare metrics across countries and regions
- **Platform Breakdown**: Market share visualization for search engines

## Technologies Used

- **Tableau**: Dashboard development and interactive visualization
- **Python**: Data collection and preprocessing
- **Statistical Analysis**: Trend analysis and correlation studies

## Key Findings

The analysis revealed significant regional disparities in internet access and notable shifts in search engine preferences, particularly in emerging markets.
